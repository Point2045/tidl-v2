05 — Voice & Copy

Direct, not clinical

Principles

✓
Direct, not clinical — "Your health, delivered" not "Optimizing patient outcomes"
✓
Confident, not arrogant — "Clinically-proven" not "The best treatment available"
✓
Warm, not casual — "We're here for you" not "Hey bestie"
✓
Specific, not vague — "$20/mo" not "Affordable pricing"
✓
Active, not passive — "Start your visit" not "A visit can be started"

06 — Design Rules

Do's & don'ts

✓ Do

✓
Use gold sparingly — CTAs, accents, hover states only
✓
Use Cormorant Garamond for headings and prices
✓
Use Inter for all UI text, buttons, labels
✓
Use sharp edges (0 radius) as primary design language
✓
Use dark sections for contrast and visual rhythm
✓
Use italic gold for emphasis words in headings
✓
Maintain WCAG AA contrast ratios (4.5:1 body, 3:1 large)

✗ Don't

✗
Use gold as body text color
✗
Use Cormorant Garamond below 1rem
✗
Use Inter for display headings
✗
Round primary component corners (architectural feel)
✗
Use stock medical imagery (stethoscopes, white coats)
✗
Use "Submit", "Click Here", or "Buy Now" as CTAs
✗
Mix TIDL visual identity with Point 2045 branding
